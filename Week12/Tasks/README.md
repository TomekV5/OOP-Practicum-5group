# Задачи


### Задача 1
Реализирайте клас **Toy**
 - име
 - цена
 - производител
 - индекс на износеност (New, Used)
и методи:
 - makeSound()
 - operator~, който променя индекса на износеност на Used
 - operator!, който променя индекс на износеност на New

Реализирайте класовете **DogToy**, **TrainToy**, **SquishyToy**, **CarToy** с подходящи промени по методите

### Задача 2
Реализирайте клас **Shape**, представляващ геометричнаа фигура. Има следните методи:
 - getPerimeter()
 - getArea()
 - isPointInside()

Поддържа следните фигури: **Triangle**, **Rectangle**, **Square**, **Circle**

### Задача 3
Реализирайте система за онлайн магазин:

Клас **User**;
 - име
 - ид
 - парола
 - години
 - наличност
 - seeBasket()

Имаме три типа User-и: **GoldUser**, **SilverUser**, **BronzeUser**

За всяка цена за продукт - *GoldUser* има 20% отстъпка, *SilverUser* има 10%, а *BronzeUser* няма


Клас **Product**:
 - цена
 - производител
 - име
 - ид
 - getSize(), връщаш подходящо съобщение спрямо типа продукт (подходяща мерна единица за броене)

Имаме два вида продукти: *FoodProduct*, *DrinkProducct*

Реализирайте клас **FoodMarket**, който управлява тези потребители и продукти, позволяващ закупуване на продукти, харесване на продукти, управляване на наличността за потребителите и зареждане на магазина
